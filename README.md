## 🇬🇧 English

### 📖 About the Project
DocuMind AI is an enterprise Retrieval-Augmented Generation (RAG) platform performing document chunking, vector embedding search, and contextual Q&A with source page attribution.

### ✨ Key Features
- 📄 Multi-format document ingestion (PDF, TXT, MD)
- 🧮 Sliding-window vector chunking & indexing
- 🔍 Semantic similarity search with Vector Store
- 📌 Precise page & source snippet attribution
- 🛡️ Enterprise security (20MB upload ceiling & input sanitization)

### 💻 Getting Started
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## 🇫🇷 Français

### 📖 À propos du projet
DocuMind AI est une plateforme RAG (Retrieval-Augmented Generation) d'entreprise effectuant le découpage de documents, la recherche par embeddings vectoriels et des réponses contextualisées avec attribution des sources.

### ✨ Fonctionnalités clés
- 📄 Ingestion multi-format de documents (PDF, TXT, MD)
- 🧮 Découpage vectoriel à fenêtre glissante & indexation
- 🔍 Recherche par similitude sémantique via Vector Store
- 📌 Attribution précise de la page et du passage source
- 🛡️ Normes de sécurité (Limite 20 Mo & assainissement des entrées)
